2. Presentasikan hasil aplikasi serta jelaskan implementasi dari poin-poin kriteria di atas menggunakan aplikasi https://www.loom.com

Ketentuan Pengerjaan

- Berikan panduan cara install. 
- Sertakan Dummy Data / Faker jika ada data yg diperlukan untuk menjalankan aplikasi di dokumentasi (Misal untuk akun login dummy).
- Sertakan .env atau credential jika ada aplikasi third party
- Jika diperlukan deploy ke platform hosting (Vercel, Netifly, Heroku, dll).
